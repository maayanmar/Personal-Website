The App

Reader View

A continuous, infinite-scroll reading experience for the entire Tanakh. Verses load progressively as you scroll, with commentaries lazy-loaded on demand.

  • Infinite-scroll with lazy-loaded commentaries
  • Fisheye progress rail for chapter navigation
  • Clean typography optimized for Hebrew text
  • Book & chapter navigation bar
Pardes View

A deep-study layout that places the verse front and center, surrounded by a grid of classical commentaries with color-coded source badges.

  • Verse displayed prominently at the top
  • 3 tabs: Commentaries, Community, Data
  • Multi-column commentary grid from 37+ sources
  • Color-coded badges per commentary source

Development Methodology

Built primarily with Claude Code using MCP servers for Supabase management and Cloudflare deployment. Frontend prototyping done with Lovable, then refined and extended via Claude Code.

An AI-assisted development workflow — from database schema to production deployment.

GCP Enterprise → Supabase + Cloudflare

Migrated the entire stack from Google Cloud Platform Enterprise to Supabase (backend, auth, database) and Cloudflare Pages (hosting, CDN) — significantly reducing costs while improving maintainability and security.
