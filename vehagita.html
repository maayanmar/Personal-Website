Key Features

  • Infinite-scroll reader — continuous verse stream with lazy-loaded commentaries
  • Full-text search — search across verses and commentaries
  • Fisheye progress rail — visual chapter navigation with zoom effect
  • Multiple commentary layouts — side-by-side, inline, and stacked views
  • Personal insights (pirushim) — Tiptap rich text editor for writing and sharing
  • Google OAuth — sign in to save progress and insights
  • Offline caching — IndexedDB for previously viewed content
  • Light & dark themes — system-aware with manual toggle
  • Performance refactor — optimized rendering and bundle splitting
  • Migration savings — moved from Vercel to Cloudflare Pages, cutting hosting costs
